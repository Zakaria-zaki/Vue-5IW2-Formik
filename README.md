# Formik

```bash
$ npm install formik --save
```

## Usage
### Linux
 - we have Makefile

```bash
# start docker-compose
$ make start 

# stop docker-compose
$ make stop 
```
### Windows or MacOS
```bash
# start docker-compose
$ docker-compose up --build
```

```bash
# stop docker-compose
$ docker-compose down
```
